Books

[Problem Statement]

Develop a Backend Application in Java Spring Boot/Hibernate.

[Minimum Requirement]

  • Use the Books API to fetch Book details and dump it into the database.

  • GET request on “/” should return the same JSON. 

[Plus Point]

  • GET request on “/:bookID” should return the individual book details.

Example

“/33385” should return 

{

    "bookID": 33385,

    "title": "Das Haus mit der grünen Tür (Varg Veum  #1)",

    "authors": "Gunnar Staalesen",

    "average_rating": 3.34,

    "isbn": "344245008X",

    "language_code": "ger",

    "ratings_count": 46,

    "price": 609

  }

 

[Advanced]

Implement Fuzzy Search - A GET request on “/search/:term” should return all books containing the title term

Example

“/search/Harry”

 

[Guide]

Books API: https://s3-ap-southeast-1.amazonaws.com/he-public-data/books8f8fe52.json

Note: Use this API to Fetch Book details.
